# 📊 Dashboard de Vendas

## 🎯 Objetivo

Este projeto tem como objetivo analisar dados de vendas e apresentar insights estratégicos por meio de um dashboard interativo.

---

## 🚀 Tecnologias utilizadas

* Python
* Pandas
* Matplotlib
* Seaborn
* Streamlit

---

## 📊 Funcionalidades

* Filtro por região, categoria e período
* Indicadores de faturamento
* Análise temporal de vendas
* Ranking de produtos
* Visualizações interativas

---

## ▶️ Como executar

```bash
pip install streamlit pandas matplotlib seaborn
python -m streamlit run dashboard/app.py
```

---

## 📌 Principais insights

* Tendência de crescimento nas vendas
* Quedas sazonais no mês de janeiro
* Distribuição equilibrada da receita entre produtos
